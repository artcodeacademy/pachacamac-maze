Pachacamac Maze

In this project we will create a maze inspired in the archaelogical site of Pachacamac. The objective is to collect all the treasures to win. Click the example to play the game:

Click here

1) Create a new project, draw a backdrop that you can use as a maze, make sure to only use 2 different colors; one for the background and the other one for the walls of the maze.

2) Now choose a character to go through the maze, make it small enough so it can pass through the maze without touching the walls and place it where you want the maze to start.

3) When you press the green flag make your character start in the position you chose.

4) Use the event blocks to move your character to the right when you press the right arrow key.

Hint: Remember the x represents the horizontal movement (left - right) and the y the vertical movement (up - down).

5) Now do the same thing for all the directions, left, up, and down.

Hint: Some directions use negative numbers, do you know why?

6) If your character touches the maze walls then make it return to the start of the maze.

Hint: Look through the sensing blocks for one that might help you accomplish this.

7) Choose 3 objects so they can be your treasure and place them in different parts of the maze where your character can come to collect them. When the green flag is pressed made the objects show.

8) Now let's create a variable called "Treasures". In the code we will add that if your character touches the treasure it disappears and the variable decreases by one.

9) Create another backdrop to tell the player they have won by discovering all of the treasures at Pachacamac. Be creative!

10) When you press the green flag make the maze backdrop show and set the "Treasures" variable to 3.

11) Add code so that if your character collects all the treasures your backdrop changes to the winning backdrop.

Bonus: Try personalizing the maze by making it more difficult or adding more treasures.

*** Important: Always remember to save your project and share it when you're done.
